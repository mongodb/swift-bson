  • encode(_:)

    Encodes an instance of the Encodable Type T into Data representing canonical or relaxed extended JSON. The value of self.mode will determine which format is used. If it is not set explicitly, relaxed will be used.

    See also

    https://docs.mongodb.com/manual/reference/mongodb-extended-json/

    EncodingError if the value is corrupt or cannot be converted to valid ExtendedJSON.
